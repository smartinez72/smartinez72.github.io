Class: Everytime a class takes place, the subject, time, teacher, and discussion topic are encoded. This makes it possible to recreate the wole class period.
Component Type Number of Bits/Range Desription/Comments
Room Number ASCII 80 bits (0-10 chars) This describes the number or name of classroom the class takes place.
Subject ASCII 160 bits (0-20 chars) Name of Class subject. 20 characters is enough.
Time ASCII 72 bits (9 chars) Describes the time class starts(hh:mm a.m./p.m.). ASCII used to add colon(:) to separate hours and minutes and to add A.M or P.M
Teacher ASCII 120 bits (0-15 chars) Describes Name of Teacher. 32 Chars should be enogh
Discussion Topic ASCII 4000 bits (0-500 chars) 500 chars should be enough to give a briefly summarize class discussion
